Detailed Description

This class lets you get fake information that looks real.

Example: 

echo 'Name: ' . Faker::name() . "\n";
echo 'Phone: ' . Faker::phone_number() . "\n";

Output: 

Name: Abbigail Vandervort
Phone: 1-202-479-9161
